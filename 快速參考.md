## ❌ 故障排除

### User 無法登入
```bash
# 刪除舊資料
del secure_web_data.json

# 重新啟動
python secure_web_system.py
```

### 端口被佔用
```bash
# 查找並結束
netstat -ano | findstr :5000
taskkill /F /PID <PID>
```
