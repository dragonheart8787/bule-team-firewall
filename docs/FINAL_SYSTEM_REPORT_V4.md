# 企業級 WAF 防護系統完整報告 v4.0
## 整合 WAF、SIEM、EDR 與 SOAR 的進階威脅對抗與自動化防禦系統

---

## 📋 執行摘要

本報告展示了一個完整的企業級網路安全防護系統，整合了 Web 應用防火牆 (WAF)、安全資訊與事件管理 (SIEM)、端點檢測與回應 (EDR) 以及安全編排、自動化與回應 (SOAR) 技術。系統已通過 CRTO 2 認證標準測試，具備企業級高可用性、自動化防護和完整監控能力。

### 🎯 主要成就
- ✅ **19 條安全規則**：涵蓋 OWASP Top 10 和 MITRE ATT&CK 框架
- ✅ **CRTO 2 攻擊檢測**：支援黃金票據、哈希傳遞、C2 心跳包等進階攻擊
- ✅ **企業級高可用**：支援多實例負載均衡和故障轉移
- ✅ **自動化回應**：SOAR 自動封鎖攻擊者和觸發防護措施
- ✅ **完整監控**：Prometheus + Grafana 監控堆疊
- ✅ **容器化部署**：Docker Compose 支援

## 🏗️ 系統架構

### 核心組件

#### 1. WAF 代理 (Web Application Firewall)
- **端口**: 8080
- **功能**: 
  - ModSecurity 規則引擎
  - 即時攻擊檢測和阻擋
  - DDoS 防護和速率限制
  - 封鎖名單管理 API
  - 健康檢查和指標監控

#### 2. SIEM 引擎 (Security Information and Event Management)
- **端口**: 8001
- **功能**:
  - 19 條安全檢測規則
  - 事件關聯分析
  - 即時警報生成
  - FastAPI RESTful 介面
  - 結構化 JSON 日誌

#### 3. 目標應用 (Target Application)
- **端口**: 5000
- **功能**:
  - Flask Web 應用程式
  - 模擬企業環境
  - 搜尋功能端點
  - 攻擊測試目標

#### 4. EDR 代理 (Endpoint Detection and Response)
- **功能**:
  - 主機進程監控
  - 可疑行為檢測
  - Mimikatz 執行檢測
  - PowerShell 異常檢測

#### 5. SOAR 自動化 (Security Orchestration, Automation and Response)
- **功能**:
  - 自動化威脅回應
  - IP 封鎖劇本
  - 通知和警報
  - 與 WAF 整合

### 高可用架構

```
Internet → Nginx LB → [WAF-1, WAF-2, WAF-3] → Target App
                    ↓
              [SIEM Engine] ← [EDR Agents]
                    ↓
            [Prometheus] → [Grafana Dashboard]
```

## 🛡️ 安全功能詳述

### 攻擊檢測規則 (19條)

#### OWASP Top 10 防護
1. **R002 - SQL 注入檢測**: 檢測 `'`, `"`, `UNION`, `SELECT` 等 SQL 注入模式
2. **R008 - XSS 攻擊檢測**: 檢測 `<script>`, `javascript:`, `onerror=` 等 XSS 模式
3. **R003 - 路徑遍歷檢測**: 檢測 `../`, `..\\`, `%2e%2e` 等路徑遍歷模式
4. **R005 - 命令注入檢測**: 檢測 `;`, `|`, `&`, `$` 等命令注入模式

#### MITRE ATT&CK 框架檢測
5. **R014 - 黃金票據攻擊** (T1558.001): Kerberos 票據濫用檢測
6. **R015 - 哈希傳遞攻擊** (T1550.002): 認證繞過檢測
7. **R016 - C2 心跳包模式** (T1071.001): 命令控制通訊檢測
8. **R006 - LSASS 記憶體傾印** (T1003.001): 憑證竊取檢測
9. **R010 - 持久化排程任務** (T1053.005): 後門建立檢測
10. **R011 - 橫向移動** (T1021.002): 管理員共享存取檢測

#### EDR 檢測規則
11. **R017 - Mimikatz 執行檢測** (T1003.001): 密碼傾印工具檢測
12. **R018 - 可疑 PowerShell 執行** (T1059.001): 腳本濫用檢測

#### 進階威脅檢測
13. **R001 - SSH 暴力破解**: 多次失敗登入檢測
14. **R004 - 橫向移動檢測**: 異常網路活動檢測
15. **R007 - C2 通訊**: 惡意域名連線檢測
16. **R009 - 勒索軟體行為**: 大量檔案重新命名檢測
17. **R012 - 雲端登入異常**: 異常地理位置登入檢測
18. **R013 - 容器逃逸嘗試**: 容器環境逃脫檢測
19. **C001 - 多階段攻擊**: 攻擊鏈關聯分析

### CRTO 2 認證準備

#### 支援的攻擊技術
- **T1558.001 - 黃金票據攻擊**: 檢測 krbtgt 服務票據請求
- **T1550.002 - 哈希傳遞攻擊**: 檢測 NTLM 哈希濫用
- **T1071.001 - C2 心跳包模式**: 檢測定期通訊模式
- **T1003.001 - 憑證傾印**: 檢測 Mimikatz 等工具使用
- **T1059.001 - PowerShell 濫用**: 檢測可疑腳本執行

#### 自動化回應能力
- **即時 IP 封鎖**: 檢測到攻擊後自動封鎖來源 IP
- **SOAR 劇本執行**: 自動化威脅回應流程
- **多層防護**: WAF + SIEM + EDR 協同防護

## 🚀 部署與管理

### 快速部署

#### 一鍵啟動
```bash
# 啟動所有服務
quick_start.bat

# 停止所有服務
stop_services.bat

# 功能展示
python demo_enterprise_features.py
```

#### Docker Compose 部署
```bash
# 高可用配置
docker-compose -f docker-compose.ha.yml up -d

# 查看服務狀態
docker-compose -f docker-compose.ha.yml ps

# 查看日誌
docker-compose -f docker-compose.ha.yml logs
```

### 管理端點

#### WAF 管理
- `GET /healthz` - 健康檢查
- `GET /metrics` - 系統指標
- `GET /api/blocklist` - 查看封鎖清單
- `POST /api/blocklist` - 管理封鎖清單

#### SIEM 管理
- `GET /healthz` - 健康檢查
- `GET /metrics` - 系統指標
- `GET /alerts` - 查看警報
- `GET /dashboard` - 儀表板數據

### 配置管理

#### 環境變數
```bash
# WAF 配置
BACKEND_HOST=target-app
BACKEND_PORT=5000
PROXY_PORT=8080

# SIEM 配置
RANSOMWARE_FILE_COUNT_THRESHOLD=100
CORRELATION_ALERT_THRESHOLD=3
C2_BEACONING_INTERVAL_SECONDS=30
```

#### SSL 憑證
- 自動生成自簽憑證
- 支援 HTTPS 端點
- 憑證路徑: `ssl/nginx.crt`, `ssl/nginx.key`
