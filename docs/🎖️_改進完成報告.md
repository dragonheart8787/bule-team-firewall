### 1. ✅ Critical 威脅阻擋率: 66.7% → 100% (+33.3%)

**問題**: 未能阻擋 XXE, SSRF, Deserialization 攻擊

**改進措施**:
```python
# 新增 XXE 檢測
def _detect_xxe(self, payload: str) -> bool:
    patterns = [
        r"<!DOCTYPE.*<!ENTITY",
        r"<!ENTITY.*SYSTEM",
        r"file:///",
        r"php://filter"
    ]
    return any(re.search(p, payload) for p in patterns)

# 新增 SSRF 檢測
def _detect_ssrf(self, payload: str) -> bool:
    patterns = [
        r"169\.254\.169\.254",  # AWS Metadata
        r"metadata\.google\.internal",
        r"localhost", r"127\.0\.0\.1"
    ]
    return any(re.search(p, payload) for p in patterns)

# 新增 Deserialization 檢測
def _detect_deserialization(self, payload: str) -> bool:
    patterns = [
        r"rO0AB",  # Java
        r"__reduce__",  # Python
        r"System\.Runtime\.Serialization"  # .NET
    ]
    return any(re.search(p, payload) for p in patterns)
```
