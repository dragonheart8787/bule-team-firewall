# 籃隊防禦系統 - 功能與能力總覽

> 完整整理專案所有模組的功能與能力

---

## 一、核心防禦模組

### 1.1 自製防火牆系統 (`national_defense_firewall.py`)

| 能力類別 | 功能 | 說明 |
|---------|------|------|
| **深度封包檢測 (DPI)** | 9 種攻擊檢測 | SQL 注入、XSS、XXE、SSRF、反序列化、命令注入、LDAP 注入、JNDI/Log4Shell、Shellcode |
| **入侵防禦 (IPS)** | 特徵庫檢測 | 12 種攻擊工具簽名（Metasploit、Mimikatz、Cobalt Strike、Empire 等） |
| **異常檢測** | 流量異常 | 大封包、高連線率、非標準端口 |
| **Anti-APT** | APT 行為 | 長時間連線、非工作時間、大流量、Beaconing、橫向移動 |
| **Zero-Day 防護** | 多維度分析 | 行為分析、啟發式、沙箱、機器學習 |
| **SSL/TLS 檢測** | 憑證安全 | 自簽憑證、過期、弱加密、JA3 指紋 |
| **勒索軟體** | 行為檢測 | 大量檔案修改、副檔名變更、勒索信、Shadow Copy 刪除 |
| **資料外洩防護 (DLP)** | 8 類敏感資料 | 信用卡、身分證、私鑰、API Key、原始碼、密碼、機密關鍵字 |
| **虛擬補丁** | CVE 防護 | Log4Shell、EternalBlue、BlueKeep 等 CVE |


### 1.2 Kill Chain 檢測器 (`kill_chain_detector.py`)

| 階段 | 名稱 | 檢測方法 |
|-----|------|---------|
| 1 | 偵察 (Reconnaissance) | 連線率、DNS 查詢、nmap 特徵 |
| 2 | 武器化 (Weaponization) | 惡意副檔名、熵值、PE 異常、惡意內容 |
| 3 | 傳遞 (Delivery) | 釣魚關鍵字、惡意 URL、DPI |
| 4 | 漏洞利用 (Exploitation) | DPI + 虛擬補丁 + Zero-Day |
| 5 | 安裝 (Installation) | 持久化、檔案變更、註冊表、服務、排程、DLL 注入 |
| 6 | 命令與控制 (C2) | Beaconing、規律間隔、非標準埠、自訂加密 |
| 7 | 目標達成 (Actions) | Anti-APT + 資料外洩 + 破壞行為 |

## 二、SOAR 與自動化

### 2.1 SOAR 編排引擎 (`soar_playbooks.py`)

| Playbook | 功能 | 說明 |
|----------|------|------|
| **isolate_host** | 主機隔離 | 驗證主機、阻斷網路、停止可疑進程、收集證據、通知 SOC |
| **block_ip** | IP 封鎖 | 驗證 IP、防火牆規則、WAF 黑名單、SIEM 日誌、排程解封 |
| **quarantine_file** | 檔案隔離 | 驗證檔案、計算雜湊、移至隔離區、提交沙箱、更新 IoC |
| **revoke_credentials** | 憑證撤銷 | 驗證使用者、終止連線、撤銷憑證、產生臨時密碼、通知使用者 |
| **restore_service** | 服務恢復 | 檢查服務狀態、停止/清理、重啟、從備份恢復、故障轉移 |


### 2.2 紅隊持續演練 (`red_team_ci_system.py`)

| 攻擊場景 | 說明 |
|---------|------|
| Web 攻擊鏈 | SQL Injection → XSS → Path Traversal |
| 暴力破解 | 多次登入失敗嘗試 |
| 應用層 DDoS | 高頻請求洪水 |
| 命令注入 | 系統命令執行嘗試 |
| APT 多階段 | 複雜攻擊鏈 TTPs |
