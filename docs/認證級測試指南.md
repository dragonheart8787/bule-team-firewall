### 方法 2: 執行單獨的認證場景

#### 場景 1: SANS BTL2 - 藍隊二級認證

**測試內容**:
```
場景: Web 伺服器遭受 SQL 注入攻擊
任務:
  1. 偵測攻擊 (20 分)
  2. 隔離受影響主機 (30 分)
  3. 收集證據 (30 分)
  4. 生成事件報告 (20 分)
  
時間限制: 15 分鐘
及格分數: 70 分
```

**執行方法**:
```python
from certification_practice_tests import CertificationPracticeTests

tester = CertificationPracticeTests()
result = tester.btl2_scenario()

print(f"分數: {result['score']}/100")
print(f"等級: {result['grade']}")
print(f"MTTR: {result['mttr']:.2f} 秒")
```

**評分標準**:
- 事件創建: 5 分
- 攻擊偵測: 20 分
- 證據收集: 30 分
- SOAR 響應: 30 分
- 事件報告: 15 分
- MTTR 獎勵: 0-10 分（< 1 分鐘 +10 分）

**及格要求**: ≥ 70 分
