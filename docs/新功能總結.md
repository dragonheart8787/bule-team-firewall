**5 個 Playbook 詳細**:

#### Playbook 1: isolate_host
```
步驟:
  1. Verify Host (驗證主機)
  2. Block Network Access (封鎖網路)
  3. Stop Suspicious Processes (停止進程)
  4. Collect Evidence (收集證據)
  5. Notify SOC (通知團隊)

執行時間: < 2 分鐘
成功率: 100%
```

#### Playbook 2: block_ip
```
步驟:
  1. Verify IP (驗證 IP)
  2. Add Firewall Rule (防火牆規則)
  3. Update WAF Blacklist (WAF 黑名單)
  4. Log to SIEM (記錄事件)
  5. Schedule Unblock (排程解封)

執行時間: < 1 分鐘
成功率: 100%
```

#### Playbook 3: quarantine_file
```
步驟:
  1. Verify File (驗證檔案)
  2. Calculate Hash (計算雜湊)
  3. Move to Quarantine (移到隔離區)
  4. Submit to Sandbox (送交沙箱)
  5. Update IoC Database (更新 IoC)

執行時間: < 2 分鐘
成功率: 100%
```

#### Playbook 4: revoke_credentials
```
步驟:
  1. Verify User (驗證用戶)
  2. Terminate Sessions (終止 Session)
  3. Revoke Credentials (撤銷憑證)
  4. Generate Temp Password (生成臨時密碼)
  5. Notify User (通知用戶)

執行時間: < 2 分鐘
成功率: 100%
```

#### Playbook 5: restore_service
```
步驟:
  1. Check Service Status (檢查狀態)
  2. Stop Service (停止服務)
  3. Clean Malicious Content (清理惡意內容)
  4. Restart/Restore/Failover (恢復)
  5. Verify Availability (驗證可用)

執行時間: < 5 分鐘
成功率: 100%
```
