## 🎯 手動測試

如果您想手動測試攻擊防護：

### SQL 注入測試
```
1. 訪問登入頁面
2. 用戶名: admin' OR '1'='1
3. 點擊登入
4. 觀察: "檢測到可疑請求"
```

### XSS 測試
```
1. 用戶名: <script>alert(1)</script>
2. 點擊登入
3. 觀察: "檢測到可疑請求"
```

### DDoS 測試
```
1. 快速刷新頁面 20 次
2. 觀察: "請求速率超過限制"
```

### 中央伺服器驗證
```
1. 以 admin 登入
2. 查看資料、修改資料
3. 查看「中央伺服器傳輸記錄」
4. 訪問 http://127.0.0.1:9000
5. 確認所有操作都已傳輸
```
