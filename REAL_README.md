## 🚀 快速開始

### 系統需求
- Python 3.8 或更高版本
- Windows 10/11 或 Linux/macOS
- 至少 4GB RAM
- 至少 1GB 可用磁碟空間

### 安裝步驟

1. **下載系統**
   ```bash
   git clone <repository-url>
   cd 籃隊防禦系統
   ```

2. **安裝依賴套件**
   ```bash
   pip install cryptography psutil requests pyyaml
   ```

3. **啟動系統**
   
   **Windows:**
   ```cmd
   start_real_system.bat
   ```
   
   **Linux/macOS:**
   ```bash
   ./start_real_system.sh
   ```
   
   **或直接運行:**
   ```bash
   python real_main.py
   ```

### 測試系統
```bash
python test_real_system.py
```

## ⚙️ 真實系統配置

### 加密配置
```yaml
crypto:
  key_rotation_interval: 86400  # 24小時
  max_key_usage: 1000000
  default_key_lifetime: 365  # 天
  encryption_algorithms:
    - "AES-256-GCM"
    - "RSA-4096"
    - "ED25519"
    - "X25519"
```

### 威脅獵殺配置
```yaml
threat_hunting:
  monitoring_interval: 30  # 秒
  threat_intelligence_update: 3600  # 秒
  anomaly_threshold: 0.7
  detection_methods:
    - "signature"
    - "anomaly"
    - "behavioral"
    - "network"
    - "log_analysis"
```

### 零信任配置
```yaml
zero_trust:
  trust_decay_rate: 0.1
  verification_interval: 300  # 秒
  policy_evaluation_timeout: 5  # 秒
  device_verification:
    fingerprint_check: true
    security_status_check: true
    update_status_check: true
```
