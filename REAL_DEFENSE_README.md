# 🛡️ 真實軍事級防禦系統

## 系統概述

這是一個真實的軍事級防禦系統，具備完整的網路安全防護能力，包括：

- **真實網路監控** - 實時封包捕獲和分析
- **真實威脅檢測** - YARA 規則、行為分析、PE 分析
- **真實事件回應** - 自動隔離、證據收集、回應策略
- **真實數位鑑識** - 系統證據收集、時間線分析
- **真實惡意程式分析** - 靜態/動態分析、沙箱環境
- **真實滲透測試** - 漏洞掃描、利用測試、後滲透

## 🚀 快速開始

### 1. 安裝依賴

```bash
pip install -r requirements.txt
```

### 2. 啟動系統

**Windows:**
```bash
start_real_defense.bat
```

**Linux/Mac:**
```bash
python real_military_defense_system.py
```

### 3. 系統狀態

系統啟動後會顯示：
- ✅ 各防禦模組狀態
- 🛡️ 防禦能力摘要
- 📊 實時監控數據

## 🔧 系統架構

### 核心模組

1. **網路監控模組** (`real_network_monitor.py`)
   - 原始套接字封包捕獲
   - 實時流量分析
   - DDoS 攻擊檢測
   - 可疑連接監控

2. **威脅檢測模組** (`real_threat_detection.py`)
   - YARA 規則引擎
   - PE 文件分析
   - 行為異常檢測
   - 記憶體監控

3. **事件回應模組** (`real_incident_response.py`)
   - 自動威脅隔離
   - 證據收集
   - 回應策略執行
   - 事件升級

4. **數位鑑識模組** (`real_digital_forensics.py`)
   - 系統證據收集
   - 文件時間線分析
   - 網路證據分析
   - 行為模式識別

5. **惡意程式分析模組** (`real_malware_analysis.py`)
   - 靜態分析
   - 動態分析
   - 沙箱環境
   - 行為監控

6. **滲透測試模組** (`real_penetration_testing.py`)
   - 網路偵察
   - 漏洞掃描
   - 利用測試
   - 後滲透活動
