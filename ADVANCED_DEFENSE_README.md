# 🛡️ 軍事級進階防禦系統

## 系統概述

軍事級進階防禦系統是一個綜合性的安全防護平台，整合了多層防禦技術，提供企業級的安全保護能力。系統採用模組化架構，支援零信任架構、網路檢測與回應、工控/IoT防禦、AI威脅獵捕、威脅情報整合、紅藍對抗可視化和風險量化等核心功能。

## 🎯 核心功能

### 1. 零信任架構 (Zero Trust Architecture)
- **IAM/MFA 整合**: 身份與存取管理、多因子認證
- **NAC 系統**: 網路存取控制、裝置合規性評估
- **微分段**: 網路分段、流量隔離、政策執行
- **持續驗證**: 動態信任評估、風險評分

### 2. 網路檢測與回應 (NDR)
- **Zeek/Suricata 整合**: 網路流量分析、協議解析
- **C2 Beaconing 檢測**: 命令與控制通訊識別
- **DNS 隧道檢測**: 隱匿通訊通道發現
- **東西向流量分析**: 內部網路流量監控

### 3. 工控/IoT 防禦
- **協議監控**: Modbus、DNP3、CAN Bus 等工控協議
- **OT/SCADA 防護**: 專用工控環境安全
- **裝置發現**: 自動識別工控設備
- **異常檢測**: 工控環境威脅識別

### 4. AI 威脅獵捕
- **機器學習模型**: 異常檢測、行為分析
- **UEBA**: 用戶與實體行為分析
- **威脅模式識別**: 攻擊技術分類
- **自動化獵捕**: AI 驅動威脅發現

### 5. 威脅情報整合
- **STIX/TAXII 支援**: 標準化威脅情報格式
- **IoC 管理**: 威脅指標收集與分析
- **自動化回應**: 基於情報的自動防護
- **威脅關聯**: 多源情報整合分析

### 6. 紅藍對抗可視化
- **MITRE ATT&CK**: 攻擊技術映射
- **Kill Chain 分析**: 攻擊鏈路分析
- **覆蓋率評估**: 防禦能力評估
- **紅藍隊比較**: 攻防效果對比

### 7. 風險量化
- **FAIR 模型**: 因子分析資訊風險
- **財務影響分析**: 量化風險成本
- **風險優先級**: 自動化風險排序
- **緩解建議**: 基於風險的防護建議

## 🚀 快速開始

### 環境要求
- Python 3.8+
- Windows 10/11 或 Linux
- 8GB+ RAM
- 10GB+ 可用磁碟空間

### 安裝步驟

1. **克隆專案**
```bash
git clone <repository-url>
cd military-advanced-defense-system
```

2. **安裝依賴**
```bash
pip install -r requirements.txt
```

3. **啟動系統**
```bash
# Windows
start_advanced_defense.bat

# Linux
python military_advanced_defense_system.py
```

### 配置說明

系統使用 `military_advanced_defense_system.py` 作為主程式，支援以下配置選項：

- **分析類型**: 選擇要執行的防禦分析類型
- **時間範圍**: 設定分析的時間窗口
- **健康監控**: 啟用系統健康狀態監控
