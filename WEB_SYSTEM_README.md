## 🛡️ 安全功能展示

### 1. 密碼輸入框（✅ 已實現）

```html
<!-- 密碼自動隱藏 -->
<input type="password" placeholder="請輸入密碼">

顯示為: ********************
```

### 2. 中央伺服器資料傳輸（✅ 已實現）

所有操作都會自動傳輸到中央伺服器：

```
用戶操作 → 加密 → 傳輸 → 記錄
                ↓
         中央伺服器
```

傳輸的事件：
- ✅ 登入/登出事件
- ✅ 資料訪問事件
- ✅ 資料修改事件
- ✅ 安全警報事件
- ✅ IP 封鎖/解封事件

### 3. 中間層攻擊防護（✅ 已實現）

防護類型：

| 攻擊 | 檢測 | 動作 |
|------|------|------|
| **SQL 注入** | union, select, drop... | 阻止 + 記錄 |
| **XSS** | &lt;script&gt;, javascript:... | 阻止 + 記錄 |
| **路徑遍歷** | ../, ..\ | 阻止 + 記錄 |
| **命令注入** | \|, ;, \`, $... | 阻止 + 記錄 |

### 4. DDoS 防禦（✅ 已實現）

多層防護：

```
第一層: 全局速率限制 (100 req/min)
第二層: 端點速率限制 (20 req/min)
第三層: 自動 IP 封鎖
第四層: 威脅等級調整
```
