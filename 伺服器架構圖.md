**端口**: `5000`

**職責**:

**主要組件**:

#### A. WAF (Web Application Firewall) 模組
```python
class WAFProtection:
    def check_request(self, request):
        # SQL 注入檢測
        if self._detect_sql_injection(request):
            return block_request("SQL_INJECTION")
        
        # XSS 攻擊檢測
        if self._detect_xss(request):
            return block_request("XSS")
        
        # 路徑遍歷檢測
        if self._detect_path_traversal(request):
            return block_request("PATH_TRAVERSAL")
        
        # 命令注入檢測
        if self._detect_command_injection(request):
            return block_request("COMMAND_INJECTION")
        
        return allow_request()
```

**檢測模式**:
```
SQL 注入: 'or', 'union', 'select', 'drop', 'insert', 'update', 'delete'
XSS: '<script', 'javascript:', 'onerror', 'onload', '<svg', '<iframe'
路徑遍歷: '../', '..\\', '%2e%2e/', 'etc/passwd', 'windows/system32'
命令注入: ';', '|', '`', '$', 'ls', 'cat', 'whoami', 'dir'
```

#### B. 認證模組
```python
class AuthenticationModule:
    def __init__(self):
        self.users = {
            "admin": {
                "password_hash": pbkdf2_hmac('sha256', ...),
                "role": "admin",
                "failed_attempts": 0,
                "locked_until": None
            },
            "user": {
                "password_hash": pbkdf2_hmac('sha256', ...),
                "role": "user",
                "failed_attempts": 0,
                "locked_until": None
            }
        }
    
    def login(self, username, password):
        # 檢查帳號鎖定
        if self._is_locked(username):
            return {"error": "帳號已被鎖定"}
        
        # 驗證密碼
        if self._verify_password(username, password):
            self._reset_failed_attempts(username)
            return {"success": True, "role": user["role"]}
        else:
            self._increment_failed_attempts(username)
            if user["failed_attempts"] >= 3:
                self._lock_account(username, duration=300)
            return {"error": "密碼錯誤"}
```

**安全特性**:
- ✅ PBKDF2-HMAC-SHA256 密碼雜湊（100,000 迭代）
- ✅ 帳號鎖定機制（3 次失敗 → 鎖定 300 秒）
- ✅ CSRF Token 保護
- ✅ Secure Session Cookie
- ✅ 密碼強度要求

#### C. DDoS 防護模組
```python
class DDoSProtection:
    def __init__(self):
        self.rate_limits = {
            "global": {
                "requests": 100,
                "window": 60  # 100 req/min
            },
            "per_endpoint": {
                "requests": 20,
                "window": 60  # 20 req/min per endpoint
            },
            "per_ip": {
                "requests": 50,
                "window": 60  # 50 req/min per IP
            }
        }
        self.blocked_ips = {}
    
    def check_rate_limit(self, request):
        ip = request.remote_addr
        endpoint = request.endpoint
        
        if self._is_ip_blocked(ip):
            return {"blocked": True, "reason": "IP_BLOCKED"}
        
        if self._exceeds_global_limit():
            return {"blocked": True, "reason": "GLOBAL_RATE_LIMIT"}
        
        if self._exceeds_endpoint_limit(endpoint):
            return {"blocked": True, "reason": "ENDPOINT_RATE_LIMIT"}
        
        if self._exceeds_ip_limit(ip):
            self._block_ip(ip, duration=600)  # 封鎖 10 分鐘
            return {"blocked": True, "reason": "IP_RATE_LIMIT"}
        
        return {"blocked": False}
```

**速率限制**:
```
全局: 100 請求/分鐘
端點: 20 請求/分鐘/端點
IP: 50 請求/分鐘/IP
IP 封鎖: 10 分鐘
```

#### D. 資料傳輸模組
```python
class CentralServerTransmitter:
    def __init__(self):
        self.api_key = "SECRET_API_KEY"
        self.central_server_url = "http://127.0.0.1:9000"
    
    def transmit_event(self, event_type, data):
        # 加密資料
        encrypted_data = self._encrypt_data(data)
        
        # 計算校驗和
        checksum = self._calculate_checksum(encrypted_data)
        
        # 構建傳輸包
        payload = {
            "id": str(uuid.uuid4()),
            "timestamp": datetime.now().isoformat(),
            "user": session.get('username', 'system'),
            "data_type": event_type,
            "data": encrypted_data,
            "checksum": checksum,
            "status": "SUCCESS"
        }
        
        # 發送到中央伺服器
        response = requests.post(
            f"{self.central_server_url}/api/transmit",
            json=payload,
            headers={
                "X-API-Key": self.api_key,
                "Content-Type": "application/json"
            },
            timeout=5
        )
        
        return response.status_code == 200
    
    def _encrypt_data(self, data):
        # SHA-256 加密
        return hashlib.sha256(data.encode()).hexdigest()
    
    def _calculate_checksum(self, data):
        # MD5 校驗和
        return hashlib.md5(data.encode()).hexdigest()
```

**傳輸事件類型**:
```
• LOGIN_EVENT - 用戶登入
• LOGOUT_EVENT - 用戶登出
• DATA_ACCESS - 資料訪問
• DATA_MODIFICATION - 資料修改
• SECURITY_ALERT - 安全警報
• IP_BLOCKED - IP 封鎖
• IP_UNBLOCKED - IP 解封
```
