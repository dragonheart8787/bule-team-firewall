## 🛠️ 故障排除

### 問題 1: User 無法登入

**現象**: 輸入 `user / User@2025` 後顯示密碼錯誤

**原因**: 舊的資料檔案使用了不同的 SALT

**解決方法**:
```bash
# 1. 停止所有服務
taskkill /F /IM python.exe

# 2. 刪除舊的資料檔案
del secure_web_data.json

# 3. 重新啟動系統
python secure_web_system.py
```

系統會自動生成新的資料檔案，使用固定的 SALT。

### 問題 2: 看不到 SHA-256 加密資料

**現象**: 儀表板中傳輸記錄沒有顯示完整的 SHA-256

**解決方法**:
1. 確保以 `admin` 身份登入（User 看不到傳輸記錄）
2. 確保中央伺服器正在運行
3. 執行一些操作觸發事件（登入、查看資料、修改資料）
4. 刷新頁面

### 問題 3: 中央伺服器連接失敗

**現象**: 傳輸狀態顯示 OFFLINE

**解決方法**:
```bash
# 1. 啟動中央伺服器
python central_server.py

# 2. 確認服務運行
# 訪問 http://127.0.0.1:9000
# 應該看到 "Central Server is running."
```
